프로젝트 목표 및 기술 스택

본 프로젝트의 목표는 협업 흐름을 자동화하여 디스코드, GitHub, Notion 등의 다양한 플랫폼 간에 효율적인 정보 공유와 관리 시스템을 제공하는 것입니다. 이를 위해 다음과 같은 기술 스택과 도구들이 사용됩니다.

기술 스택

1. Python 3.12

  • Discord-py-interactions: 디스코드 챗봇 개발을 위한 라이브러리입니다.
  • SQLAlchemy: ORM을 통해 데이터베이스와의 상호작용을 용이하게 합니다.
  • FastAPI: 빠르고 효율적인 웹 서버 개발을 위한 Python 프레임워크입니다.
  • Alembic: 데이터베이스 자동 업데이트 및 관리 도구입니다.

2. APIs

  • Discord API: 디스코드 챗봇과의 연동을 위한 API입니다.
  • Notion API: Notion 페이지에서 정보를 입출력하고, 유저 정보를 취득하는 데 사용됩니다.
  • GitHub API: 유저 프로필, 리포지토리 정보, 활동 정보를 가져오기 위한 API입니다.

3. 기타 도구

  • Nginx: Reverse Proxy로 사용되어 트래픽을 효과적으로 분배합니다.
  • Docker: 컨테이너화된 환경에서 프로젝트를 테스트하고 배포하기 위한 도구입니다.
  • MySQL: 관계형 데이터베이스 관리 시스템으로 프로젝트의 데이터를 관리합니다.
배포 환경 및 협업 도구

1. 배포 환경

  • Github: 소스 코드 배포 및 개발을 위한 버전 관리 시스템입니다.
  • Docker: 컨테이너 환경에서의 배포와 테스트를 위한 환경을 제공합니다.
  • Nginx: 리버스 프록시 서버로, 효율적인 트래픽 분배 및 서버 운영을 돕습니다.

2. 협업 도구

  • Notion: 프로젝트 회의, 개발 진행 상황 추적, 문서화를 위한 협업 도구입니다.
  • Discord: 팀 커뮤니케이션 및 실시간 앱 테스트를 위한 플랫폼입니다.
  • GitHub: 개발자들 간의 소스 코드 관리와 협업을 위한 주요 플랫폼입니다.

이 프로젝트는 오픈소스로 제공되며, 보안 등의 이유로 민감한 정보를 보호하려는 목적으로 누구나 사용할 수 있도록 배포됩니다. 프로젝트는 디스코드 봇 형태로 제공될 예정입니다.
